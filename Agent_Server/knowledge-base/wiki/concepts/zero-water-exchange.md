# 零換水（Zero Water Exchange）

## Definition
零換水是指在水產養殖生產週期中，幾乎不進行外部水交換的經營模式。其目標是將營養物質在系統內部完全回收利用，減少對外部水源的依賴並消除污水排放。

## Implementation in BFT
- **廢物循環**：利用生物絮技術將代謝廢物轉化為生物質。
- **生物安全**：切斷外部病原體進入的途徑，防止病原傳播。
- **環境保護**：避免向周邊環境排放富營養化廢水。
