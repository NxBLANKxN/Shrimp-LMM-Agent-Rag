# 抗菌肽（Antimicrobial Peptides）

## Definition
抗菌肽（AMPs）是一類短鏈蛋白質，具有能破壞病原微生物（如細菌、真菌、病毒）細胞膜或干擾其內部代謝而導致其死亡的能力，是無脊椎動物先天免疫系統的重要組成部分。

## Identification in M. rosenbergii
- **預測方法**：利用 AI 模型（如 AI4AMP）結合訊號肽（Signal Peptide）篩選。
- **已知類型**：包含 crustin, Histone H2A, CgUbiquitin, kazal2, Techylectin-5B, MrH4 等。
- **潛在數量**：在淡水長臂大蝦中預測出約 232 筆潛在 AMPs 序列。
