# Nitrification

**定義**: 硝化作用 (Nitrification) 是一種由化學自營細菌 (Chemoautotrophic bacteria) 驅動的兩步生物氧化過程，將有毒的氨轉化為相對無毒的硝酸鹽。

## 過程
1. **氨氧化 (Ammonia Oxidation)**: 
   - $\text{NH}_3 / \text{NH}_4^+ \rightarrow \text{NO}_2^-$ (亞硝酸鹽)
   - 主要細菌: *Nitrosomonas*, *Nitrosococcus*, *Nitrospira*, *Nitrosolobus*, *Nitrosovibrio*。
2. **亞硝酸鹽氧化 (Nitrite Oxidation)**: 
   - $\text{NO}_2^- \rightarrow \text{NO}_3^-$ (硝酸鹽)
   - 主要細菌: *Nitrobacter*, *Nitrococcus*, *Nitrospira*, *Nitrospina*。

## 在 BFT 中的角色
- **毒性控制**: 有效降低總氨氮 (TAN) 和亞硝酸鹽濃度，防止養殖動物中毒。
- **鹼度消耗**: 硝化過程會消耗碳酸氫根 ($\text{HCO}_3^-$)，導致 pH 值下降，因此需定期添加鹼性物質 (如 $\text{NaHCO}_3$)。
- **氧需求**: 硝化細菌是絕對好氧菌，需要充足的溶氧。

## 與異營同化的區別
- **硝化作用**: 產生硝酸鹽，固體產量極低，速度較慢。
- **異營同化**: 將氮納入細菌生質，產生大量生物絮固體，速度較快。
