# 飼料轉換率（Feed Conversion Ratio）

## Definition
飼料轉換率（FCR）是一個衡量養殖動物將飼料轉化為體重增加之效率的指標。計算方式為：$\text{FCR} = \frac{\text{總投餵飼料量}}{\text{生物量增加量}}$。

## Significance
- **經濟指標**：FCR 越低，代表飼料利用率越高，生產成本越低。
- **環境指標**：低 FCR 意味著產生較少的廢棄物排放到環境中，環境成本較低。
- **物種差異**：在蝦類中，南美白對蝦通常被認為比黑虎蝦具有更好的 FCR，適合大規模集約化生產。
