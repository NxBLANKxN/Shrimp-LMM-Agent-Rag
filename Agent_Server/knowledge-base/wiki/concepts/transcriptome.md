# 轉錄體（Transcriptome）

## Definition
轉錄體是指在特定時間、特定環境或特定組織中，細胞內所有轉錄出的 RNA 分子的集合。透過 RNA-seq 技術與從頭組裝（de novo assembly），可以建構出物種的轉錄基因體，用於研究基因表達模式與功能註解。

## Application in Shrimp Research
- **功能分析**：在缺乏完整基因體（Genome）的情況下，轉錄體可提供蛋白質編碼序列的資訊。
- **差異表達分析**：比較健康與感染（如 MrNV, Vibrio harveyi）個體的轉錄體，找出關鍵免疫基因。
- **基因發現**：用於鑑別新型抗菌肽（AMPs）或激素家族（如 CHH）。
