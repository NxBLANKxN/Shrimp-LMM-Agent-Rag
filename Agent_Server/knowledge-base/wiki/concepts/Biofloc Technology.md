# 概念定義：生物絮技術 (Biofloc Technology, BFT)

**定義:** 生物絮技術 (BFT) 是一種在水產養殖場域內建立和維持穩定微生物聚落 (microbial community) 的技術。它通過極少換水、強氧化和調整水域中的碳氮比來實現。

**核心機制與生態服務:**
*   **營養循環:** 將養殖過程產生的廢物（如過剩的營養物質）同化成微生物生物質，豐富天然食物鏈基礎。
*   **污染控制:** 將游離的潛在污染源轉化或固化成有用的生物資源，排除環境毒性。
*   **生物安全:** 透過培育多樣化的微生物聚落，提高對病原體的競爭與排斥，改善生物安全。

**經濟效益與優點:**
*   **降低成本:** 顯著減少水質維護成本（如過濾費用）、節約水資源、降低飼料成本。
*   **提升生產力:** 提升活存率和成長率，增加產量。
*   **環境永續:** 減少用水和排水，降低每公斤產品的生態足跡。

**微生物組成:**
*   **細菌:** 異營細菌（利用有機化合物作為碳源，如分解糞便、未攝食飼料）和化學自營細菌（如硝化細菌，將氨和亞硝酸鹽分解成硝酸鹽）。
*   **真菌:** 主要為酵母菌和黴菌，可作為益生菌補充劑，並提供高蛋白質營養。
*   **微藻:** 主要為光合自營生物（如綠藻、矽藻、藍綠藻），它們同化氨/銨和硝酸鹽，並產生氧氣。

**應用趨勢:**
*   BFT 系統的發展是應對水資源有限和環境污染的現代化解決方案，尤其適用於零換水或少換水系統。
*   BFT 系統的組成是動態的，會隨著碳氮比和管理方式的變化而轉變（從自營群聚轉變為異營群聚）。
