# 異營細菌（Heterotrophic Bacteria）

## Definition
異營細菌是利用有機碳作為能量與碳源的微生物。在生物絮技術中，異營細菌可在碳源充足時快速吸收水體中的氨氮，將其轉化為微生物生質，降低毒性氮累積。

## Key Points
- 需要足夠可利用碳源，因此與 [[carbon-nitrogen-ratio]] 管理高度相關。
- 是 [[biofloc-technology]] 中快速氮移除與生物絮形成的重要參與者。
- 過度增殖可能提高耗氧量，因此需要良好曝氣與溶氧監控。
