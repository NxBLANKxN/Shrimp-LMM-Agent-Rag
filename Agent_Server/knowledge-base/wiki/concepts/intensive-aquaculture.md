# 高密度養殖（Intensive Aquaculture）

## Definition
高密度養殖（或稱集約式養殖）是指在有限的養殖面積中，透過提高放養密度來追求最大化單位產量的養殖模式。這種模式通常依賴人工飼料投餵與強化的水質管理（如打氣、換水）。

## Challenges
- **水質壓力**：高生物量（Biomass）導致代謝廢物（如氨氮）快速累積，溶氧消耗劇增。
- **疾病風險**：高密度環境易導致個體間壓力增加，降低免疫力，且病原體在密集族群中傳播速度更快。
- **資源競爭**：個體間對空間與食物的競爭增加，可能導致生長不均或互相殘食。

## Mitigation Strategies
- **間捕（Partial Harvest）**：透過分批捕撈大個體來降低生物負荷。
- **強化水質管理**：增加溶氧、提高換水率、使用生物製劑淨化水質。
