# 概念定義：轉錄組學 (Transcriptomics)

**定義:** 轉錄組學是研究生物體中所有轉錄產物（即 RNA）的學科。它通過分析 RNA 序列，可以了解生物體在特定條件下（如疾病感染、環境壓力）的基因表達模式和功能。

**核心技術與流程:**
1.  **序列獲取:** 從公開資料庫 (如 NCBI SRA) 獲取 RNA-seq 短讀序列。
2.  **從頭組裝 (De novo Assembly):** 將短讀序列組合成完整的轉錄基因體 (Transcriptome)，這是從零開始重建基因序列的過程。
3.  **組裝評估:** 使用工具（如 BUSCO）評估組裝體的完整度和品質。
4.  **功能性註解:** 預測蛋白質序列，並使用各種資料庫（如 Pfam, GO, NR）進行功能註解，確定蛋白質的功能區塊和生物學過程。

**應用價值:**
*   **基因功能解析:** 確定基因在特定生理狀態下的表達情況。
*   **疾病機制研究:** 識別與病毒感染（如諾達病毒、白點症病毒）相關的基因變化。
*   **育種與改良:** 為智慧育種提供基因層級的數據基礎。
*   **數據平台:** 建立線上分析系統 (如 MOLAS)，方便研究者進行多組學數據的比較分析。
