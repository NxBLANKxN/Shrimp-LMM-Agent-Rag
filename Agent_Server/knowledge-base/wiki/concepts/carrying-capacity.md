# 生產載量（Carrying Capacity）

## Definition
生產載量是指在特定的環境條件下，一個養殖系統（如蝦池）能夠支持的生物最大量，且不導致環境品質惡化或生物健康受損。

## Dynamics in Shrimp Ponds
- **隨時間遞減**：隨著養殖時間增加，代謝廢物累積與水質惡化，最適生產載量通常會逐漸減少。
- **影響因素**：溶氧量、氨氮濃度、溫度及水體自淨能力。
- **管理手段**：透過間捕（Partial Harvest）可以人為降低現存量，使其重新回到生產載量的安全範圍內。
