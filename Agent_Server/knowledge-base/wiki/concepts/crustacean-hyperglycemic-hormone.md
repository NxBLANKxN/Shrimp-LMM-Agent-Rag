# 甲殼類高血糖素（Crustacean Hyperglycemic Hormone）

## Definition
甲殼類高血糖素（CHH）是一個結構相關但功能多樣的蛋白質家族，主要負責調節甲殼類動物的血糖濃度及其他生理代謝過程。

## Comparative Analysis
- **多樣性差異**：對蝦科（Penaeidae，如白蝦、草蝦）的 CHH 數量與多樣性顯著高於長臂蝦科（Palaemonidae，如淡水長臂大蝦、日本沼蝦）。
- **基因體複製影響**：儘管日本沼蝦經歷了全基因體複製（WGD），但其 CHH 數量並未隨之增加。
