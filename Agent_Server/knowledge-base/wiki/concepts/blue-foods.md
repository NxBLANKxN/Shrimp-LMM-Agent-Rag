# 藍色食物（Blue Foods）

## Definition
藍色食物是指所有來自海洋、湖泊、河流等水域的食物，包括魚類、甲殼類、軟體動物、藻類以及水生植物。這類食物被認為是全球食物系統中增長最快且對營養安全至關重要的部分。

## Importance
- **營養價值**：提供高品質動物蛋白質、微量營養素（如維生素 B12）及必需脂肪酸（Omega-3）。
- **環境效率**：相較於陸地動物，許多水產養殖物種具有更低的飼料轉換率（FCR），意味著較低的環境成本。
- **經濟貢獻**：全球約 10% 的人口依賴藍色食物獲取收入。
