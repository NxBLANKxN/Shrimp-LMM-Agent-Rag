# 生物絮技術（Biofloc Technology）

## Definition
生物絮技術（BFT）是一種水產養殖技術，透過在養殖場域內建立並維護穩定的微生物聚落（Microbial Community），將養殖過程中產生的廢物（如氨氮、磷酸鹽）同化為微生物生質。這不僅能淨化水質，還能為養殖動物提供天然食物來源，實現極少換水或零換水養殖。

## Core Mechanisms
- **微生物聚落**：由細菌、微藻、真菌及浮游動物組成的懸浮聚合物。
- **碳氮比（C:N Ratio）管理**：透過添加碳源（如糖類、澱粉）來促進異營細菌的生長，使其快速吸收氨氮。
- **強氧化**：提供充足的溶氧以維持好氧微生物的活性並保持絮團懸浮。

## Benefits
- **環境永續**：大幅減少用水量與污水排放，降低生態足跡。
- **生物安全**：減少換水降低病原入侵風險；多樣化微生物聚落透過競爭排斥抑制病原體（如弧菌）。
- **經濟效益**：降低飼料轉換率（FCR）與水處理成本。
