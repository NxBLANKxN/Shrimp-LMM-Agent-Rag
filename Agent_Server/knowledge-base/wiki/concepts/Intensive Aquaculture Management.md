# 概念定義：集約式養殖管理 (Intensive Aquaculture Management)

**定義:** 集約式養殖管理是指在有限的養殖面積內，透過提高生物密度、精細化管理和引入特定技術來最大化產量的養殖模式。

**核心挑戰:**
*   **水質惡化:** 高密度養殖導致蝦體排泄物、飼料殘餌腐敗，造成水質劇變。
*   **環境壓力:** 氨態氮 (NH3-N) 增加、溶氧量 (DO) 降低、pH 值波動等，對蝦類造成緊迫 (stress)。
*   **疾病風險:** 環境壓力會削弱蝦體的免疫力，增加病毒性及細菌性疾病爆發的風險。

**關鍵管理策略:**
*   **間捕 (Partial Harvest):** 在養殖週期中進行分批收穫，這是一種有效的管理技術，用於：
    *   維持蝦池的生產載量 (carrying capacity)。
    *   減少養殖風險，避免因過高生物量導致的水質崩潰。
    *   提高產量和經濟效益，並使小個體快速成長至上市規格。
*   **水質控制:** 嚴格監測和控制溫度、鹽度、溶氧、pH 值和氨氮等關鍵環境因子。
*   **生物安全:** 增強蝦體的免疫力，預防病原體感染。
