# 蝦類物種比較（Shrimp Species Comparison）

## Definition
蝦類物種比較是針對不同養殖蝦種在生長週期、市場定位、飼料轉換率、疾病風險與集約化適性上的差異進行整理。此概念有助於評估養殖策略與供應鏈韌性。

## Key Points
- 南美白對蝦通常適合高產量與集約化生產，並與較低 [[fcr]] 相關。
- 黑虎蝦常被視為高端市場產品，生長週期較長且系統集約程度通常較低。
- 物種選擇是 [[resilient-aquaculture]] 的重要策略之一。
