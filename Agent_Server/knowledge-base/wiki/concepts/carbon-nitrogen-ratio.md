# 碳氮比（Carbon-Nitrogen Ratio）

## Definition
碳氮比是指水產養殖系統中可利用碳源與氮源的比例。在生物絮技術（BFT）中，調整碳氮比是控制水質、促進異營細菌生長以去除氨氮的核心手段。

## Role in BFT
- **促進異營生長**：異營細菌需要碳源作為能量與結構組成。當碳源充足時，細菌會快速吸收水中的氨氮來合成蛋白質。
- **氨氮去除**：高碳氮比有利於將氨氮直接轉化為微生物生質，而非僅依賴較慢的硝化作用。
