# 硝化細菌（Nitrifying Bacteria）

## Definition
硝化細菌是一類化學自營微生物，可將氨轉化為亞硝酸鹽，再進一步轉化為硝酸鹽。在養殖水體中，硝化作用是降低氨毒性的重要途徑，但通常比異營細菌同化氮的速度慢。

## Key Points
- 與 [[heterotrophic-bacteria]] 共同構成生物絮系統中的氮轉化路徑。
- 對溶氧、pH 與環境穩定性敏感。
- 在 [[biofloc-technology]] 中協助控制氨與亞硝酸鹽累積。
