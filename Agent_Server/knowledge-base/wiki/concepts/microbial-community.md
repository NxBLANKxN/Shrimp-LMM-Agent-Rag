# 微生物聚落（Microbial Community）

## Definition
在生物絮技術（BFT）中，微生物聚落是指由多種微小生物（包括細菌、微藻、真菌、原生動物等）與非生命物質（如多醣、蛋白質、碎屑）透過物理化學力聚集而成的獨立懸浮聚合物。

## Composition
- **細菌**：主導成分，包含自營硝化細菌與異營細菌。
- **微藻**：提供光合作用與初步氮同化。
- **真菌**：如酵母菌，增加營養價值並可能具有益生作用。
- **浮游動物**：啃食微生物，將能量傳遞至更高營養階層。

## Dynamics
- **演替**：系統通常從光合自營菌（綠色水）主導轉向異營細菌（棕色水）主導。
- **共生關係**：細菌與微藻之間存在互惠關係，共同控制氨氮與亞硝酸鹽。
