# Recirculating Aquaculture System (RAS)

**Definition:** An indoor, closed-loop aquaculture system that filters and recycles water, allowing for high-density production with minimal water exchange and high control over environmental conditions.

**Key Features:**
- Uses biological filtration, disinfection, and aeration.
- Enables production closer to urban centers (reducing logistics costs).
- Higher initial capital (CAPEX) and operational (OPEX) costs compared to pond systems.
- Part of the "super-intensive" production typology.
