# 概念定義：[[生物絮技術 (BFT)]]

**定義：**
[[生物絮技術 (BFT)]] 是一種在水產養殖場域內，透過建立和維持穩定的微生物聚落（生物絮），來實現養殖廢物循環利用、水質淨化和生物安全保障的先進養殖系統。
**核心特徵：**
1.  **低換水率：** 系統設計允許極少甚至零換水。
2.  **高氧化性：** 需維持強氧化環境以支持微生物活動。
3.  **碳氮比控制：** 透過調整水域中的碳氮比來優化微生物群落的生長。
**生態服務：**
*   **營養循環：** 將養殖廢物中的過剩營養物質（如氮、磷）同化成微生物生物質，豐富天然食物鏈。
*   **水質淨化：** 微生物聚落（特別是硝化細菌和異營菌）能有效去除水中的有毒物質（如氨、亞硝酸鹽）。
*   **生物安全：** 多樣化的微生物群落能提高對病原體的競爭和排斥能力。
**經濟效益：**
*   降低飼料轉換率 (FCR) 和飼料成本。
*   減少水資源消耗和水質維護成本。
**發展背景：**
[[生物絮技術 (BFT)]] 是應對密集養殖帶來的環境負擔（如富營養化、水資源短缺）的解決方案，是水產養殖的「藍色革命」。
