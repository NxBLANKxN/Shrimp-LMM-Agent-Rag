# 間捕（Partial Harvest）

## Definition
間捕是指在養殖週期中，不一次性全部收成，而是根據蝦隻生長情況，分批將達到上市規格的大個體捕撈出去，留下較小個體繼續飼養的管理技術。

## Objectives & Benefits
- **降低生物負荷**：減少池中總生物量（Biomass），緩解對水質的壓力（如降低氨氮、提高溶氧）。
- **維護生產載量**：使環境能持續支持剩餘生物的健康生長。
- **優化生長**：減少競爭，使小個體能快速成長，提高整體體型均一度。
- **降低風險**：在疾病爆發前或水質惡化前提前獲利，減少全池死亡風險。
- **提高經濟效益**：增加總產量，並透過獲取較大體型個體提高單價。
