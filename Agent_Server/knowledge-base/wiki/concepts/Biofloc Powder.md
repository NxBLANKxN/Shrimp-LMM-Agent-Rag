# Biofloc Powder

**定義**: 生物絮粉 (Biofloc Powder) 是將 BFT 系統中產生的生物絮固體經過乾燥、碾磨處理後製成的粉末，可用作水產養殖飼料的補充或替代品。

## 營養價值
- **高蛋白**: 含有由細菌和微藻構成的高蛋白質。
- **酶活性**: 含有能改善養殖動物消化道的酶。
- **必需營養素**: 包含多種氨基酸，但通常缺乏必需脂肪酸 (需透過調整 C:N 比或添加發酵劑來改善)。

## 應用效果
- **替代魚粉**: 研究顯示，使用 15.6% - 25% 的生物絮粉替代魚粉或大豆蛋白，可顯著提高蝦類的生長速度與存活率。
- **提高免疫**: 改善腸道微生物組，增強動物的抗病能力。

## 限制
- **灰分含量**: 過高比例的替代 (如 > 66%) 可能因灰分過高而降低生長率。
- **加工損耗**: 乾燥過程可能影響部分生物活性成分。
