# 韌性養殖（Resilient Aquaculture）

## Definition
韌性養殖是指建立能夠在面對已知（如週期性疾病）或未知（如突發極端天氣、新型病毒）干擾時，仍能維持基本生產功能並能迅速恢復到穩定狀態的養殖系統。

## Key Strategies for Resilience
- **物種選擇**：選擇對環境適應力強、生長週期短（降低經濟風險）的物種（如南美白對蝦）。
- **系統多樣化**：避免過度依賴單一生產模式或單一物種。
- **協作治理**：透過全球合作解決資源短缺與疾病傳播問題。
- **環境監控**：建立預警系統以應對氣候變化帶來的壓力。
