# 多維體分析（Multi-omics Analysis）

## Definition
多維體分析是指整合多個層級的生物分子數據（如基因體 Genome、轉錄體 Transcriptome、蛋白質體 Proteome、代謝體 Metabolome）進行綜合分析的方法，旨在從系統生物學的角度理解生物體的生理功能與病理機制。

## Implementation Example (MOLAS)
- **MOLAS 平台**：整合了淡水長臂大蝦的基因體、轉錄體、蛋白質體與功能註解結果，提供圖像化分析環境。
- **應用**：可進行不同組織間的基因差異表現量研究。
