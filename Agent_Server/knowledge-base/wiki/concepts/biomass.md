# 現存量（Biomass）

## Definition
現存量是指養殖池中生物總重量，通常由放養數量、存活率與平均體重估算。現存量過高會提高耗氧與代謝廢物累積，進而壓縮系統的安全生產空間。

## Key Points
- 與 [[carrying-capacity]] 直接相關，是判斷是否需要間捕的重要依據。
- 在 [[intensive-aquaculture]] 中，現存量快速上升會增加氨氮、亞硝酸鹽與溶氧壓力。
- [[partial-harvest]] 可透過降低現存量來恢復水質緩衝能力。
