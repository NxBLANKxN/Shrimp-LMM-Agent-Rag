# 親緣關係（Phylogeny）

## Definition
親緣關係描述物種或基因之間的演化關聯，通常透過序列比對、系統發育樹與分化時間估計來推論。在蝦類研究中，親緣分析可協助比較不同科別或物種的基因家族差異。

## Key Points
- 可用於解釋 [[crustacean-hyperglycemic-hormone]] 等基因家族在不同蝦類中的多樣性差異。
- 與 [[transcriptome]] 和多維體資料整合後，可補足缺乏完整基因體物種的演化背景。
- 淡水長臂大蝦與日本沼蝦的分化時間是來源頁記錄的關鍵案例。
