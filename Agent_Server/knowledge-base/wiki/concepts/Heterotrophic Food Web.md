# Heterotrophic Food Web

**定義**: 異營養食物網 (Heterotrophic Food Web) 是指以異營微生物 (如細菌、真菌) 為基礎的營養傳遞系統，與以植物浮游生物為基礎的自營食物網相對。

## 結構
在 BFT 系統中，異營養食物網的傳遞路徑如下：
**有機碳/氮 $\rightarrow$ 異營細菌 $\rightarrow$ 原生動物 (纖毛蟲等) $\rightarrow$ 小型甲殼類 (輪蟲、橈足類) $\rightarrow$ 養殖動物 (魚、蝦)**

## 特點
- **廢物轉化**: 將溶解的有機物和無機氮直接轉化為高蛋白質的微生物生質。
- **高效能**: 能夠在低光照或高密度環境下運作，不受光合作用限制。
- **營養增強**: 微生物在轉化過程中可累積必需脂肪酸和氨基酸。
