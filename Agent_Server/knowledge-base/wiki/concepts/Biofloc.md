# Biofloc

**定義**: 生物絮 (Biofloc) 是指在水產養殖系統中，由細菌、微藻、真菌、浮游動物 (如原生動物、輪蟲、橈足類) 與非生命物質 (如殘餌、糞便) 透過物理化學力及胞外聚合物 (EPS) 聚集而成的鬆散懸浮聚合物。

## 組成
- **微生物**: 主要是變形菌門 (Proteobacteria)，包括硝化細菌與異營細菌。
- **微藻**: 如小球藻 (Chlorella) 和矽藻，提供氧氣並同化氮。
- **浮游動物**: 纖毛蟲、輪蟲、橈足類等，作為能量傳遞的仲介。

## 生態功能
- **營養回收**: 將溶解的氮、磷轉化為蛋白質和脂質。
- **天然餌料**: 養殖動物 (如蝦、魚) 可直接攝食生物絮，降低對人工飼料的依賴。
- **水質維護**: 硝化細菌將有毒氨轉化為較無毒的硝酸鹽。

## 物理特性
- **大小**: 通常在微米至數毫米之間。
- **管理**: 需透過適當的打氣與攪拌使其保持懸浮，避免在底部形成厭氧污泥。
