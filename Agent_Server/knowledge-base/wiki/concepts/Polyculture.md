# Polyculture / Co-culture

**Definition:** The practice of farming multiple aquatic species in the same environment.

**Benefits in Shrimp Aquaculture:**
- **Water Quality**: Certain species can help manage nitrogenous waste or control algae.
- **Economic Resilience**: Diversifies income streams, reducing the impact of a single crop failure.
- **Feed Efficiency**: Some co-cultured species can provide live feedstock (e.g., *Artemia*) for larvae.
- **Environmental Balance**: Creates a more stable ecosystem that can be more resilient to stressors.
