# BCf Technology

**全稱**: Biofloc-copefloc Technology (橈足蟲-生物絮技術)

**定義**: BCf 是一種結合了 BFT (生物絮技術) 與橈足蟲 (Copepod) 培養的密集式養殖技術，旨在提供高品質的天然活餌，以取代人工飼料。

## 核心流程
1. **生物絮建立**: 建立穩定的異營微生物聚落。
2. **橈足蟲培養**: 在 BFT 環境中接種並大規模培養橈足類動物。
3. **天然餵食**: 橈足蟲攝食生物絮中的細菌與微藻，再被養殖蝦類攝食。

## 優勢
- **零人工飼料**: 實現完全依賴天然餌料的生產，大幅降低飼料成本。
- **快速生長**: 提供更符合自然需求的營養，提升幼蝦的成活率與生長速度。
- **高度可持續**: 形成閉環營養循環，對環境零污染。

## 應用
主要應用於南美白對蝦 (*Litopenaeus vannamei*) 的密集式養殖。
