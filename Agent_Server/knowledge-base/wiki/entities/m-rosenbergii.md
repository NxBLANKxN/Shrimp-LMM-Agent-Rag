# 淡水長臂大蝦（Macrobrachium rosenbergii）

## Description
淡水長臂大蝦是一種全球重要且產值逐年增長的淡水養殖蝦種。

## Biological Characteristics
- **分類**：屬於長臂蝦科（Palaemonidae）。
- **基因體特徵**：相較於對蝦科（Penaeidae），其甲殼類高血糖素（CHH）家族的數量與多樣性較低。
- **親緣關係**：與日本沼蝦（*M. nipponense*）的共同祖先約出現在 94 百萬年前。

## Research Status
- **轉錄體**：已有高完整度（96.5%）的轉錄基因體組裝，包含約 20.9 萬條轉錄序列。
- **免疫系統**：具有多種抗菌肽（AMPs），包括 crustin, Histone H2A, MrH4 等。
