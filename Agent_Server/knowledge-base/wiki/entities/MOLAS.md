# MOLAS

**MOLAS** (Multi-Omics online Analysis System) is a web-based database designed to integrate and analyze multi-omics data for common Asian shrimp species.

## Features
- **Data Integration**: Stores genomic, transcriptomic, and proteomic sequences.
- **Functional Annotation**: Provides annotations for proteins, including GO terms and KEGG pathways.
- **Differential Expression Analysis**: Allows users to upload their own data and compare gene expression across different conditions or tissues.
- **Species Covered**: Includes *Macrobrachium rosenbergii*, *L. vannamei*, *P. monodon*, *F. chinensis*, *M. japonicus*, and *M. nipponense*.

## Purpose
To provide a user-friendly, graphical interface for researchers to perform comparative genomics and transcriptomics, aiding in studies of physiology and disease control.
