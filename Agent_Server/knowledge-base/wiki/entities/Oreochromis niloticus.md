# Oreochromis niloticus

**中文名**: 尼羅吳郭魚
**分類**: 魚類 $\rightarrow$ 吳郭魚屬

## BFT 相關特性
- **攝食能力**: 相比於蝦類，吳郭魚能更有效地直接消費水中的生物絮顆粒。
- **耐受力**: 能適應高密度的 BFT 系統，並在其中展現出較低的飼料轉換率 (FCR)。
- **生產力**: 在 BFT 系統中的產量通常極高 (可達 10-30 kg/m²)。

## 相關研究
- 使用光合自營微藻 (如小球藻) 的 BFT 系統可顯著提高其初期生長率與免疫反應。
