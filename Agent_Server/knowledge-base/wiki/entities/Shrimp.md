# 實體定義：蝦 (Shrimp)

**描述:** 蝦是全球最重要的蛋白質來源之一，在水產養殖中佔有極高的比例。

**相關資訊:**
*   **主要物種:** 黑虎蝦 (*Penaeus monodon*) 和太平洋白蝦 (*Litopenaeus vannamei*)，兩者佔全球養殖產量的 80%。
*   **養殖模式:**
    *   **廣泛/半集約式:** 多為小型農戶，服務於國內市場。
    *   **集約/超集約式:** 服務於國際出口，由大型產業參與。
*   **挑戰:** 疾病（如黃頭病毒）、氣候變化、飼料依賴性、環境壓力。
*   **永續性:** 蝦殼（Chitin）可作為高價值副產品用於生物技術和醫藥。
