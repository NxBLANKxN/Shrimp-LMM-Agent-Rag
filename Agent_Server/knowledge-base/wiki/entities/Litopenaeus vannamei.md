# Litopenaeus vannamei

**中文名**: 南美白對蝦 / 太平洋白蝦
**分類**: 甲殼類 $\rightarrow$ 十足目 $\rightarrow$ 對蝦屬

## BFT 相關特性
- **適應性**: 是 BFT 技術最主要的研究與商業對象，能耐受高密度的懸浮固體。
- **攝食習慣**: 具有強烈的過濾攝食與雜食特性，能有效攝食生物絮中的微生物。
- **生產模式**: 常被用於超密集式跑道養殖 (Super-intensive raceway system) 或零換水池塘養殖。

## 相關研究
- 使用 BCf 技術可實現零人工飼料養殖。
- 生物絮粉可替代部分魚粉，提高其生長率與存活率。
