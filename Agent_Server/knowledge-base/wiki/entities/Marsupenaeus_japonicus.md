# Marsupenaeus japonicus

***Marsupenaeus japonicus***, known as the Kuruma prawn, is a marine aquaculture species.

## Key Facts
- **Family**: Penaeidae.
- **Genome**: Genome size is approximately 1.7 Gb with 43 pairs of chromosomes.
- **Research**: Shows high genomic similarity (approx. 69.9%) with *L. vannamei* and *P. monodon*.
