# 淡水長臂大蝦（Macrobrachium rosenbergii）

## Description
淡水長臂大蝦是一種全球重要且產值逐年增長的淡水養殖蝦種。

## Genomic Status
- **轉錄體**：已建立高完整度（96.5%）的轉錄基因體，包含約 20.9 萬條轉錄序列。
- **蛋白質**：預測有 15.4 萬筆蛋白質序列。
- **資料庫**：相關多維體資料已整合至 MOLAS 線上分析系統。

## Biological Features
- **分類**：屬於長臂蝦科（Palaemonidae），是重要的淡水養殖蝦種。
- **免疫**：具有多樣的抗菌肽（AMPs）系統。
- **內分泌特徵**：相較於對蝦科（Penaeidae），其甲殼類高血糖素（CHH）家族的數量與多樣性較低。
- **親緣關係**：與日本沼蝦（*M. nipponense*）的共同祖先約在 94 百萬年前。
