# 實體定義：淡水長臂大蝦 (Macrobrachium rosenbergii)

**描述:** 淡水長臂大蝦是一種重要的淡水養殖蝦種，其產值在全球逐年增長，是進行基因體學和轉錄組學研究的對象。

**相關資訊:**
*   **研究重點:** 由於缺乏完整的基因體序列，本研究利用轉錄組學方法進行從頭組裝，以建構完整的轉錄基因體。
*   **研究應用:** 透過轉錄體和蛋白質體的功能性註解，研究其免疫機制、抗菌肽 (AMPs) 鑑定、以及與其他蝦種的親緣關係。
*   **生物學特徵:**
    *   **免疫系統:** 具有潛在的抗菌肽序列，可作為抗病機制的研究對象。
    *   **激素系統:** 具有甲殼類高血糖素 (CHH) 家族，用於研究物種分化和生理調節。
    *   **數據平台:** 相關的基因體、轉錄體和功能性註解數據已建構為線上資料庫 MOLAS，供研究者使用。
*   **比較基因體學:** 與白蝦、草蝦等常見養殖蝦種進行比較，以了解物種間的基因異同。
