# Litopenaeus stylirostris (藍蝦)

**Common Name**: Blue shrimp
**Type**: Marine Shrimp

## Farming Notes
- **Production**: In super-intensive systems using partial harvesting (7 harvests plus a final one), yields of up to 20 tons/ha have been reported, with individual shrimp reaching 30g.
