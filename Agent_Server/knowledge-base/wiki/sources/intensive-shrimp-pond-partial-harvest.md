# 高密度養殖蝦池間捕增產之研究

## Summary
本研究為國立臺灣大學碩士論文，探討在超高密度蝦類養殖中，如何透過「間捕」（Partial Harvest）管理技術來緩解中後期水質惡化、降低養殖風險並提高總產量。研究指出，隨著養殖時間增加，蝦池現存量（Biomass）過高會導致氨態氮增加、溶氧降低，進而抑制生長或引發疾病。透過間捕（分批捕撈大個體），可以有效維護蝦池的生產載量（Carrying Capacity），使小個體能快速成長，並提高整體經濟效益。

## Key Concepts
- [[intensive-aquaculture]]: 超高密度養殖，雖能提高單位產量但增加水質管理難度與疾病風險。
- [[partial-harvest]]: 間捕，透過分批捕撈大個體來降低生物負荷，維持水質穩定。
- [[carrying-capacity]]: 生產載量，環境能支持的生物最大量，隨水質惡化而減少。
- [[biomass]]: 現存量，池中生物的總重量，過高會導致水質劇變。

## Detailed Notes

### 1. 高密度養殖的困境
- **水質惡化**：中後期因現存量過高、排泄物與殘餌累積，導致氨態氮 $\uparrow$、溶氧 $\downarrow$。
- **負面影響**：生長緩慢、體型參差不齊、疾病叢生（尤其是病毒爆發時）、飼料浪費。
- **風險**：嚴重時會導致部分或全池死亡。

### 2. 間捕（Partial Harvest）的效果
- **維護生產載量**：減少生物負荷，使水質較易維持穩定。
- **促進成長**：間捕後，留下的較小個體因競爭減少而能快速成長至上市規格。
- **提高產量與效益**：
    - **斑節蝦 A 池**：4 個月養殖 + 間捕 $\rightarrow$ 產量達 28 噸/公頃（體型 18-24g）。
    - **斑節蝦 B 池**：3 個月養殖 + 間捕 $\rightarrow$ 產量達 10 噸/公頃（體型 7-10g）。
    - **草蝦 A 池**：2 個月養殖 + 間捕 $\rightarrow$ 總產量 7.13 噸/公頃，較未間捕增產 1.65 噸/公頃，且體型較大、價格較高。

### 3. 管理建議
- 除了增加換水、增加水車溶氧或施用藥劑/活菌外，間捕是目前最有效的健康安全與增產管理方式之一。
- 雖然間捕會增加人工與網具成本，但其帶來的增產與風險降低效益遠高於成本。
