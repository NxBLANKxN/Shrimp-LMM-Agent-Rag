# 來源摘要：全球蝦養殖的挑戰與韌性策略
**來源文件:** s41538-026-00787-7_reference.pdf
**主題:** 全球蝦養殖系統的挑戰、資源流動和五種韌性策略。
**發布狀態:** 待發表 (Article in Press)
**關鍵發現:**

**1. 產業重要性與需求:**
*   藍食（Blue Foods，來自水產養殖和漁業）對全球飲食和經濟至關重要。
*   全球藍食消費量持續增長，預計到 2050 年將再次翻倍。
*   蝦類是全球市場上最重要的蛋白質來源之一（約佔 70% 的養殖產量）。

**2. 養殖系統與物種:**
*   全球養殖主要集中在拉丁美洲和東南亞，供應高收入市場。
*   主要生產物種為黑虎蝦 (*Penaeus monodon*) 和太平洋白蝦 (*Litopenaeus vannamei*)。
*   養殖模式分為：
    *   **廣泛/半集約式 (Extensive/Semi-intensive):** 多為小型農戶，服務於國內市場。
    *   **集約/超集約式 (Intensive/Super-intensive):** 服務於國際出口，由大型產業參與。

**3. 主要挑戰與壓力源 (Stressors):**
*   **疾病與環境:** 疾病爆發（如黃頭病毒、白斑症候群病毒）、氣候變化和極端天氣。
*   **資源消耗:**
    *   飼料是主要成本和環境壓力源（約佔 70% 的溫室氣體排放和 50% 的生產成本）。
    *   蝦類是水產養殖中第二大野生魚類消費者（僅次於鮭魚）。
    *   病原體免費幼蝦供應不足。
*   **市場與供應鏈:** 供應鏈複雜，需要應對食品安全、欺詐和環境管理等挑戰。

**4. 韌性策略與趨勢 (Resilience Strategies):**
*   **技術應用:** 採用機器學習 (ML)、人工智慧 (AI)、物聯網 (IoT)、自動化、循環養殖系統 (RAS) 和共養技術。
*   **永續性:** 關注永續性、獲得相關認證、開發替代飼料來源。
*   **副產品利用:** 蝦殼（Chitin）是重要的廢棄物，可用於生物技術、生物化學、藥品和食品等產業。
*   **供應鏈透明度:** 應用區塊鏈技術進行可追溯性管理。

**結論:**
為確保全球蝦供應的永續性和韌性，產業必須在技術創新、資源管理和協作上做出改變。
