# 文獻摘要：生物絮技術在水產養殖的應用 (FER202505501)

**來源：** FER202505501.pdf
**主題：** [[生物絮技術 (BFT)]] 在水產養殖中的應用、機制與優點。
**核心內容：**
生物絮技術是一種在養殖場域內建立和維持穩定微生物聚落的系統，其核心是利用微生物聚落循環利用養殖廢物，將營養物質同化為生物資源。
**主要優點：**
1.  **生態效益：** 循環使用廢物，豐富天然食物鏈基礎，排除毒性，抑制病原體滋生，增強生物安全。
2.  **經濟效益：** 降低飼料成本、減少水質維護成本、節約水資源、提升活存率和產量。
3.  **環境效益：** 減少生態足跡，有利於可持續發展。
**技術機制：**
*   **生物絮 (Biofloc)：** 由各種微小生物（藻類、細菌、浮游動物、真菌）和非生命物質構成的懸浮聚合物。
*   **微生物組成：** 主要由異營需氧微生物（如變形菌門、擬桿菌門）、光合自營菌（微藻）和真菌組成。
*   **氮循環：** 氮的去除主要通過三種途徑：藻類光自營活動、自營細菌（硝化/脫氮）和異營菌直接同化。
**發展歷程：**
[[生物絮技術 (BFT)]] 是應對密集養殖帶來的富營養化和水資源短缺的現代技術，其概念可追溯到 20 世紀 60 年代以色列和台灣的施肥型式試驗。
**潛在風險：**
系統中可能存在有害藻華 (HAB) 和病原體（如弧菌屬 *Vibrio*），需要持續監測。
