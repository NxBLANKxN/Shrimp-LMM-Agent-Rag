# 生物絮技術在水產養殖的應用

**日期**: 2025-06-02 (Available online)
**作者**: 陳瑤湖 (國立臺灣海洋大學水產養殖學系)
**來源**: 海大漁推 55 期：01-62 (2025)
**DOI**: 10.29474/FER.202512.0106

## 關鍵發現
- **BFT 的核心價值**: 生物絮技術 (BFT) 被稱為水產養殖的「藍色革命」，其核心在於建立穩定的微生物聚落，將養殖廢物轉化為微生物生質，提供天然營養並改善水質。
- **經濟與環境效益**: 能顯著降低飼料成本 (FCR 降低)、減少換水需求 (零換水或少換水)、提升生物安全並減少環境足跡。
- **碳氮比 (C:N Ratio) 管理**: 透過調整碳氮比 (初始階段 12-20:1，維護階段 6:1) 來促進異營細菌的同化作用，有效去除氨氮。
- **BCf 技術**: 結合生物絮與橈足蟲 (Copepod) 培養的 BCf 技術可實現完全不使用人工飼料的密集式對蝦養殖。
- **關鍵管理參數**: 溶氧 (DO > 4 ppm)、溫度 (> 20°C)、總鹼度 (> 100 ppm) 以及有機顆粒的重新懸浮是維持系統穩定的關鍵。

## 詳細摘要

### 1. 生物絮技術 (BFT) 的概念與優點
BFT 旨在透過強氧化、低換水及調整碳氮比，在養殖場內建立微生物聚落。該聚落將過剩的營養物質 (氮、磷) 同化為微生物生質，豐富食物鏈基礎，並抑制病原體滋生。相較於傳統養殖，BFT 能降低水費、飼料費 (約 30% 濾水費用減少，飼料成本降低)，並透過減少換水提高生物安全性。

### 2. 生物絮的組成與動態
生物絮 (Biofloc) 是由細菌、微藻、真菌及浮游動物 (原生動物、輪蟲、橈足類、枝角類、線蟲) 與非生命物質構成的懸浮聚合物。
- **細菌**: 變形菌門 (Proteobacteria) 最為豐富，包括硝化細菌 (化學自營) 和異營養細菌。
- **微藻**: 提供氧氣並同化氮，如小球藻 (Chlorella) 和矽藻。
- **浮游動物**: 作為能量傳遞的仲介，將微生物營養轉移至養殖動物。

### 3. 生產互動與水質管理
- **氧動力學**: 微生物聚落對氧的需求極高，有時甚至超過養殖動物本身，必須具備可靠的增氧系統。
- **固體管理**: 必須防止底層厭氧污泥積累 (避免產生 H2S)，透過打氣使顆粒重新懸浮。
- **氮廢物處理**: 
    - **同化作用**: 異營細菌利用有機碳同化氨氮。
    - **硝化作用**: 硝化細菌將氨轉化為硝酸鹽。

### 4. 實務應用
- **適用物種**: 雜食性強、耐高固體含量的物種，如吳郭魚、鯉魚、鯰魚及南美白對蝦。
- **BCf 技術 (Biofloc-copefloc)**: 泰國發展的技術，利用生物絮培養橈足蟲作為天然活餌，實現零人工飼料養殖。
- **碳源選擇**: 常使用糖蜜、玉米粉、米麩等廉價副產品。

### 5. 前景與展望
- **基因組學**: 利用總體基因組學 (Metagenomics) 深入分析微生物聚落。
- **生物絮粉**: 將生物絮乾燥成粉作為飼料替代品，可提高生長率與存活率。
- **溫帶擴展**: 需研究降低加熱成本的技術以將 BFT 推廣至溫帶地區。
