# 全球韌性蝦類養殖之挑戰與策略

## Summary
本文是一篇發表於 *npj Science of Food* (2026) 的綜述文章，探討全球蝦類養殖在供應、疾病、氣候與市場永續性方面面臨的挑戰。文章將蝦類定義為「藍色食物」（Blue Foods）的典範，分析了全球生產系統（從粗放、半集約到集約與超集約系統），並提出了五項建立韌性（Resilience）的策略。重點在於如何透過協作解決資源短缺、疾病爆發與極端天氣等壓力，以確保全球安全、營養且永續的蝦類供應鏈。

## Key Concepts
- [[blue-foods]]: 來自水域的營養豐富食物，是全球食物系統中增長最快的部門。
- [[resilient-aquaculture]]: 能在已知或未知干擾（如疾病、氣候變化）中維持功能並快速恢復的養殖系統。
- [[shrimp-species-comparison]]: 
    - **南美白對蝦 (*L. vannamei*)**：佔全球產量 52%，適合大規模、高產量、低 FCR 的集約化養殖，週期短（60-90 天），經濟風險較低。
    - **黑虎蝦 (*P. monodon*)**：高端產品，體型大、口感佳，適合較小規模、低集約化的系統，週期較長（>100 天）。
- [[fcr]]: 飼料轉換率（Feed Conversion Ratio），衡量飼料轉化為生物質的效率，水產養殖通常比陸地動物更低（更高效）。

## Detailed Notes

### 1. 藍色食物（Blue Foods）的重要性
- 提供全球近 20% 的動物蛋白質。
- 關鍵微量營養素（維生素 B12）與必需脂肪酸（Omega-3）的重要來源。
- 需求預計到 2050 年將再次翻倍。

### 2. 全球蝦類養殖現況
- **市場分佈**：生產集中在拉丁美洲、東亞與東南亞；消費市場主要在北美、歐洲與日本。
- **生產模式**：
    - **粗放/半集約**：多由獨立小農經營，供應本地市場。
    - **集約/超集約**：高資本投入，由大型產業參與者主導，主要面向國際出口。
- **物種轉移**：1990 年代後期因黃頭病毒與白斑症病毒爆發，生產重心從黑虎蝦轉向南美白對蝦。

### 3. 面臨的挑戰
- **資源短缺**：飼料供應與水資源壓力。
- **疾病爆發**：病毒性疾病導致產量衰退。
- **氣候變化**：極端天氣影響生產穩定性。
- **市場永續性**：需平衡經濟增長與環境保護。
