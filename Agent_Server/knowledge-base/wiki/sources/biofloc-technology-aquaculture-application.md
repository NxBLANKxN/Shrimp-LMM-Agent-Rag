# 生物絮技術在水產養殖的應用

## Summary
本文由國立臺灣海洋大學陳瑤湖教授撰寫，全面回顧了生物絮技術（Biofloc Technology, BFT）在水產養殖中的應用。BFT 的核心在於建立並維護穩定的微生物聚落，將養殖廢物（營養物質）同化為微生物生質，提供生態服務（如水質淨化、抑制病原體）並帶來經濟效益（降低飼料成本、節水、提升產量）。文章詳細討論了生物絮的形成機制、組成（細菌、真菌、微藻）及其在系統中的動態，並強調了碳氮比管理與強氧化的重要性。

## Key Concepts
- [[biofloc-technology]]: 核心技術，透過微生物聚落實現廢物循環與生物安全。
- [[microbial-community]]: 生物絮的組成基礎，包含自營與異營微生物。
- [[carbon-nitrogen-ratio]]: BFT 管理的關鍵參數，影響異營細菌的生長與氨氮去除。
- [[zero-water-exchange]]: BFT 追求的目標，減少環境衝擊並提高生物安全。
- [[heterotrophic-bacteria]]: 負責將氨氮直接轉化為生物質的主要微生物。
- [[nitrifying-bacteria]]: 將氨轉化為硝酸鹽的化學自營細菌。

## Detailed Notes

### 1. BFT 的核心概念與優點
- **定義**：在極少換水、強氧化、調整碳氮比的條件下，維護穩定的微生物聚落。
- **生態服務**：
    - 循環利用廢物 $\rightarrow$ 微生物生質 $\rightarrow$ 天然食物鏈基礎。
    - 排除毒性 $\rightarrow$ 抑制病原體 $\rightarrow$ 提高生物安全。
- **經濟效益**：
    - 降低飼料成本（FCR 下降，如吳郭魚下降 18%）。
    - 減少水處理成本（過濾費用可減少約 30%）。
    - 節約水資源，適合水資源匱乏地區。

### 2. 生物絮（Biofloc）的組成與形成
- **物理性質**：鬆散、懸浮、飄動的聚合物（aggregates）。
- **組成**：
    - **細菌**：最豐富的組成部分。變形菌門（Proteobacteria）佔主導，包含硝化細菌（自營）與異營細菌。
    - **真菌**：如酵母菌（Saccharomyces sp.），提供高蛋白質營養，部分菌株可抑制弧菌。
    - **微藻**：光合自營生物，在系統初期主導，隨後轉向細菌主導的棕色水體。
- **形成過程**：藻類 $\rightarrow$ 細菌 $\rightarrow$ 浮游動物 $\rightarrow$ 飼料顆粒/糞便 $\rightarrow$ 絮凝（Flocculate）。

### 3. 氮轉化途徑
- **光自營**：由微藻去除。
- **自營細菌**：氨 $\rightarrow$ 亞硝酸鹽 $\rightarrow$ 硝酸鹽（硝化作用）。
- **異營細菌**：直接將氨氮同化為微生物生質（最快速且有效）。
